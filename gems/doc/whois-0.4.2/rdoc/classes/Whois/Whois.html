new(request, host_search=false)

Initialize with a request. The request must be an IPv4, or a host string

The first params now is : 

 * a string which a Ipv4, Ipv6 or a host.
 * A IPAddr instance

A second param, host_search is optionnal. By default he is false. If this value is true, a resolv host is made for know the host to this IPv4
